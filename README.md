## Setup
1. Open a Python Virtual Environment:
  ```bash
  python -m venv venv
  .\venv\Scripts\Activate.ps1 (Windows)
  source venv/bin/activate (macOS)
  ```
2. Required dependencies or libraries. You can install these with:
  ```bash
  pip install -r requirements.txt
  ```
3. Run the script
  ```bash
  python detection.py
  ```
